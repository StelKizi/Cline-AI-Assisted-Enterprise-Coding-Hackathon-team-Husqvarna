# Husqvarna Assets

This folder contains the Husqvarna reference assets extracted during UI/UX benchmarking for the `design-foundation` project.

## Purpose

These assets are stored separately so they are easy to find and reuse as:
- visual reference for landing pages
- benchmark material for slides
- source imagery for product cards and hero sections
- support material for palette, typography, and layout analysis

## Structure

### `images/hero/`
Editorial and lifestyle images suitable for hero sections.

### `images/products/`
Product-specific images used for product spotlight cards and landing pages.

### `images/reference/`
Supporting benchmark assets extracted from Husqvarna pages but not currently used directly in the landing.

### `meta/`
Non-image reference data extracted from the site:
- `palette.json`: key colors observed on the site
- `typography.json`: font families and detected weights
- `sources.json`: original page URLs and direct asset URLs
